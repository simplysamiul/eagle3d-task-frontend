# Eagle3D Task Frontend

A responsive and dynamic **Next.js frontend** for Eagle3D Task, built to manage products efficiently. The application allows users to view, add, edit, and delete products seamlessly with a modern UI.

## ðŸ”¹ Features

- **User Authentication**: Login functionality with JWT or token-based authentication.
- **Product Management**: 
  - List all products with images, price, quantity, and status.
  - Add new products via a modal form.
  - Edit existing products.
  - Delete products with confirmation alerts.
- **Responsive Design**: Works smoothly on desktop and mobile devices.
- **Instant UI Updates**: Products update instantly after add, edit, or delete actions.
- **SweetAlert2 Notifications**: Friendly success and error alerts.
